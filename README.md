## 커밋 로그
커밋 로그는 (타입: 메시지) 형식으로 되어있습니다. 아래는 커밋 타입에 대한 정보입니다.
<br><br>
Feat - 새로운 기능 추가
<br><br>
Docs - 문서 (문서 추가, 수정, 삭제)
<br><br>
Chore - 기타 변경사항 (빌드 스크립트 수정 등)
<br><br>
Fix - 버그 수정
